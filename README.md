Singularity recipe for [MATLAB Runtime 2018b](https://www.mathworks.com/products/compiler/matlab-runtime.html).

## Status

This repository is **deprecated** and receives no further updates. The base Docker image (`murphylab/matlabmcr2018b`) and MATLAB MCR 2018b are no longer supported. For current projects, please use a supported MATLAB Runtime version.

## Installation

* Install [Singularity](https://sylabs.io/docs/).
* Run the script `install.sh` included in this repository.
